# Backend routes

- **auth.ts** – OTP signup, Google check/register, get user, update profile (name/phone), update profile photo URL.
- **googleAuth.ts** – Google OAuth login/register and callback.
- **orderRoutes.ts** – Order CRUD and status.
- **shops.ts** – Shop listing and detail.
- **redeem.ts** – Redeem and balance.
- **chat.ts** – Chat messages.
- **riders.ts** – Rider-related endpoints.
- **merchantAuth.ts** – Merchant auth if used.
