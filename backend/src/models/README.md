# Backend models

Mongoose schemas for MongoDB collections.

- **User.ts** – User document: email, displayName, phone, profilePhoto, address, balance, isOnboarded, location fields, etc.
- **Order.ts** – Order document for laundry orders.
- **Shop.ts** – Shop/customer document.
- **Rider.ts**, **RiderRegistration.ts** – Rider and registration data.
- **ChatMessage.ts** – Chat messages.
- **Transaction.ts** – Wallet/balance transactions.
- **EmailOtp.ts**, **Signup.ts** – OTP and signup flow.
- **ShopRegistration.ts**, **MerchantUser.ts** – Merchant/shop registration if used.
