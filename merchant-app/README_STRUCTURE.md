# Merchant app structure

This file describes the merchant app folder structure for reference. It does not change any code.

- **app/** – Expo Router screens and layouts (if present).
- **components/** – Reusable UI components.
- **context/** – React context providers (e.g. auth).
- **config** – Base URL and API paths (if used).
- **.env.example** – Document any env vars the merchant app needs (e.g. backend URL, Supabase).

For setup and run commands, see the main README in this folder or CONTRIBUTING.md at project root.
