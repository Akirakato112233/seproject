# Rider app – app directory

Expo Router file-based routes for the rider app.

- **`(tabs)/`** – Tabs: Home/index, Active, Earning, Wallet, Account.
- **create-account.tsx**, **sign-in** – Auth entry.
- **signup/** – Rider onboarding: vehicle, documents, license, national ID, selfie, etc.
- **job.tsx** – Job detail and actions.
- **chat.tsx** – Chat with user/shop.
- **settings.tsx**, **emergency-contacts.tsx**, **linked-accounts.tsx** – Settings and related screens.
- **package/**, **vehicle-registration/**, **verify-documents/** – Signup and verification flows.
