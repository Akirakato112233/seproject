# User app – components

Reusable UI components used across the user app screens.

- **BottomSheet**, **FilterChips**, **FilterRow**, **MainFilterModal** – Filters and modals.
- **LaundryShopCard**, **LaundryShopList** – Shop listing and cards.
- **Header**, **SearchBar**, **EmptyContent** – Layout and empty states.
- **CheckboxRow**, **PriceSelector** – Form and selection controls.
- **index.ts** – Barrel export for components.
